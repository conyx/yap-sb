[boxee.scad] Parametric storage box created in OpenSCAD
-------------------------------------------------------
1) Dependencies
  - [OpenSCAD](https://github.com/openscad/openscad)
  - [BOSL2 library](https://github.com/BelfrySCAD/BOSL2)
2) Lid supports:
  - Magnets
  - Hinges
  - Latches
  - Lip
  - Slider
